## Pre-Question Checklist

1. Confirm required cluster context and node.
2. SSH into target node.
3. `sudo -i` before admin actions.
4. Implement + verify quickly.
5. `exit` root, `exit` SSH, return to base node.

## Time Strategy

- If no clear path in 30-45 seconds, flag and skip.
- Bank quick points first; return for complex multi-step tasks.
- Aim for >= 70% solved confidently, not perfection.
